# npm-user-cli

> Get user info of an npm user


## Install

```
$ npm install --global npm-user-cli
```


## Usage

```
$ npm-user --help

  Usage
    $ npm-user <username>

  Example
    $ npm-user sindresorhus
    Name: Sindre Sorhus
    Email: sindresorhus@gmail.com
    GitHub: sindresorhus
    Twitter: sindresorhus
```
